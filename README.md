# Pulse Social

A responsive dark-theme social media frontend built with React, Vite, Redux Toolkit, React Router, and Tailwind CSS.

## Stack

- React + Vite
- Redux Toolkit + RTK Query
- React Router
- Tailwind CSS
- Lucide React icons

## Features

- Feed with stories, post composer, likes, saves, comments, and tags
- Explore page with search, topics, creator results, and post grid
- Reels page with video-style cards and creator actions
- Messages with conversation list and send-message flow
- Notifications with read state
- Profile pages with follow/unfollow, edit profile, posts, and saved posts
- Mock API backed by `localStorage`, ready to replace with real backend endpoints
- Live Explore content from DummyJSON with API search and pagination

## Scripts

```bash
npm install
npm run dev
npm run build
```
